## Keyword

- All Edge cost => 1
- 두 집합(Two Partitions)
- Ford Fulkerson Solution or Simple DFS Solution
- 최대 유량 관련
- 같은 집합은 서로 연결 하지 않음. only 서로 다른 집합 끼리만 연결
