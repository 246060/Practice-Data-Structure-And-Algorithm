## 특징

- FIFO(First-in First-out)

## 구현 방법

- 필드:
    - Array or Linked-list
    - Two-Pointer
        - front
        - rear(= back)

- 메소드:
    - enQueue(= offer) : 요소 삽입
    - deQueue(= poll) : 요소 삭제
    - isEmpty: 요소 존재 유무
    - peek : 다음 나올 요소(삭제 되지 않음)
    - size : 현재 큐에 들어 잇는 요소 수

## 사용예

- 대기열
- 너비 우선 탐색(BFS)
